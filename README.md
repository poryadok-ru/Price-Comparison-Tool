# Price Comparison Tool

Инструмент для автоматического сравнения согласованных и фактических цен на товары путём сопоставления штрихкодов и кодов между шаблоном и файлами заказов.

## Что делает скрипт

Скрипт берёт Excel-шаблон с согласованными ценами и папку с файлами заказов, находит совпадения по штрихкодам/кодам товаров и заполняет в шаблоне фактические цены и разницу. Строки с расхождением цен подсвечиваются жёлтым.

### Алгоритм работы

1. **Подготовка файлов заказов** — разъединение объединённых ячеек для корректного чтения данных.
2. **Загрузка шаблона** — определение структуры столбцов по заголовкам первой строки.
3. **Построение поискового индекса** — извлечение кодов и штрихкодов из шаблона.
4. **Диагностика файлов заказов** — логирование структуры каждого файла для отладки.
5. **Поиск совпадений** — сопоставление значений из заказов с индексом шаблона.
6. **Обновление шаблона** — запись фактических цен, расчёт разницы, подсветка расхождений.
7. **Сохранение результата.**

## Требования

- Python 3.7+
- [openpyxl](https://openpyxl.readthedocs.io/)

pip install openpyxl

## Структура файлов

```
Сравнение цен/
├── шаблон.xlsx          # Шаблон с согласованными ценами
├── Заказы/              # Папка с файлами заказов (.xlsx)
│   ├── заказ_001.xlsx
│   ├── заказ_002.xlsx
│   └── ...
└── price_comparison.py  # Скрипт
```

## Формат шаблона

Первая строка шаблона должна содержать заголовки. Обязательные столбцы:

| Столбец | Описание |
|---|---|
| **Код** | Код товара |
| **Штрихкод** | Штрихкод (допускается несколько через пробел, запятую, перенос строки) |
| **Цена согласованная** | Согласованная цена |
| **Цена фактическая** | Заполняется скриптом |
| **Разница цен** | Заполняется скриптом |

## Настройка

Перед запуском укажите пути в начале скрипта:

```python
template_path = r'путь\к\шаблон.xlsx'
orders_dir = r'путь\к\папке\Заказы'
```

## Запуск

```bash
python price_comparison.py
```

Ход выполнения и результаты записываются в `price_update_log.txt` рядом со скриптом.

## Результат

После выполнения в шаблоне будут заполнены:

- **Цена фактическая** — цена из файла заказа.
- **Разница цен** — отклонение фактической цены от согласованной.
- Строки с расхождением более **0.01** подсвечены жёлтым.

## Особенности

- Штрихкоды могут быть записаны через любой разделитель (пробел, запятая, точка с запятой, перенос строки) — скрипт разберёт автоматически.
- Столбец с ценой в заказах определяется по заголовку (ключевые слова: «цена», «price», «стоимость», «сумма»).
- При наличии нескольких файлов заказов с одним и тем же товаром используется значение из последнего обработанного файла.
